Check duplicate 
Description

Given an array of integers, check if it contains duplicate element. If present then print YES else print NO.


Input
1<=T<=10

1<=N<=100000

1<=Ai<=100000


Output
output YES or NO as described for each test case.


Sample Input 1 

3
1
1
2
1 1
3
1 2 3
Sample Output 1

NO
YES
NO
